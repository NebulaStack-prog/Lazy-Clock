Часть 1. Основной документ.

1. Заголовок и базовая информация.

· Название: Lazy Clock.
· Назначение: Проект № 11. Продукт.
· Фаза проекта: Phase II.
· Технологический стек: Python, PySide6 (Qt для Python)
· Статус проекта: Полностью завершен.

2. Краткое описание проекта.

· Проект Lazy Clock – это настольные часы с аналоговым циферблатом, работающие в системном трее (свернутые в область уведомлений).
· Приложение отображает аналоговые часы с часовой, минутной и секундной стрелками на прозрачном фоне, без стандартного оконного оформления (без рамки и заголовка).
· Часы всегда находятся на рабочем столе и могут быть перемещены перетаскиванием.
· Управление приложением осуществляется через иконку в системном трее: скрытие/показ часов, режим "поверх всех окон" и выход из программы.

3. Четкие цели проекта.

· Создать настольные аналоговые часы с минималистичным дизайном.
· Реализовать отображение текущего времени с точностью до секунды.
· Обеспечить возможность перемещения часов по экрану мышью.
· Реализовать управление через системный трей (иконка в области уведомлений).
· Добавить возможность скрывать/показывать часы.
· Реализовать режим "поверх всех окон" для постоянной видимости.
· Обеспечить корректное завершение работы приложения.

4. Компоненты проекта.

· Аналоговый циферблат с тремя стрелками (часовая, минутная, секундная)
· Центральная точка в качестве оси вращения
· Прозрачный фон (без стандартного оконного оформления)
· Перетаскивание окна мышью (захват и перемещение)
· Иконка в системном трее (область уведомлений)
· Обработка сигналов завершения (SIGINT, SIGTERM)

5. Инструкция по использованию.

· Перед использованием убедитесь, что установлены необходимые библиотеки: PySide6.
· После запуска приложения в системном трее появляется иконка часов. Сами часы отображаются на рабочем столе в виде прозрачного окна с аналоговым циферблатом.
· Часы можно перетаскивать в любое место экрана, зажав левую кнопку мыши на окне часов.
· Щелчок левой кнопкой мыши по иконке трея открывает контекстное меню (примечание: в некоторых ОС используется правая кнопка).
· Двойной клик по иконке трея скрывает или показывает часы.
· В контекстном меню трея доступны следующие действия:
  · "Показать/скрыть часы" – скрывает или показывает окно часов на рабочем столе.
  · "Поверх всех окон" – включает или выключает режим отображения часов поверх всех окон.
  · "Выйти" – завершает работу приложения.
· При завершении работы через системный трей или через сигналы завершения (Ctrl+C) приложение корректно закрывается.
· Часы отображают текущее системное время с точностью до секунды. Секундная стрелка движется плавно (не дискретно).
· Циферблат не имеет фона, поскольку таковым должны выступать обои устройства для разнообразия часов, поэтому для смены внешнего вида необходимо менять обои.

Часть 2. Технический документ.

1. Цели разработки.

· Создать настольные часы с использованием библиотеки PySide6 (Qt для Python).
· Реализовать аналоговый циферблат с плавным движением стрелок.
· Отработать навыки работы с Qt-виджетами, кастомной отрисовкой через QPainter и событиями мыши.
· Реализовать управление через системный трей с контекстным меню.

2. Использованные технологии.

· Языки программирования: Python.
· Библиотеки: PySide6 (Qt для Python)

3. Архитектура проекта.

· Проект использует объектно-ориентированную архитектуру с четким разделением ответственности на основе Qt-фреймворка.
· ClockWidget – основной виджет, наследуется от QWidget. Реализует:
  · Кастомную отрисовку аналоговых часов через переопределение paintEvent()
  · Отображение трех стрелок разной длины, толщины и цвета
  · Центральную точку
  · Перетаскивание окна через обработку событий мыши (mousePressEvent, mouseMoveEvent)
  · Автоматическое обновление через QTimer (60 кадров/сек)
· MainWindow – главное окно приложения, наследуется от QMainWindow. Выполняет:
  · Создание и управление экземпляром ClockWidget
  · Создание и настройку иконки системного трея (QSystemTrayIcon)
  · Создание контекстного меню для трея с действиями
  · Обработку сигналов завершения (SIGINT, SIGTERM)
  · Управление режимом "поверх всех окон"
· Точка входа – создает экземпляр QApplication, настраивает завершение при закрытии последнего окна и запускает главный цикл событий.

4. Структура проекта.

· Проект состоит из одного основного файла: Lazy_Clock.py.
· В этом файле последовательно определены:
  · Импорты (sys, signal, math, datetime, PySide6)
  · Класс ClockWidget (наследует QWidget)
    · __init__(): инициализация размеров, флагов окна, таймера обновления
    · paintEvent(): отрисовка циферблата и стрелок
    · draw_hand(): вспомогательный метод для рисования стрелок
    · mousePressEvent(): обработка захвата окна мышью
    · mouseMoveEvent(): обработка перемещения окна
  · Класс MainWindow (наследует QMainWindow)
    · __init__(): создание часов и иконки трея
    · signal_handler(): обработка сигналов завершения
    · create_tray_icon(): создание и настройка системного трея
    · tray_clicked(): обработка кликов по иконке трея
    · toggle_clock(): скрытие/показ часов
    · toggle_top(): переключение режима "поверх всех окон"
    · quit_app(): корректное завершение приложения
  · Точка входа (if __name__ == "__main__")

5. Ключевые компоненты системы.

· Кастомная отрисовка (paintEvent) – основной компонент визуализации.
· Позиционирование стрелок – тригонометрические расчеты
· Перетаскивание окна – обработка событий мыши
· Системный трей – управление приложением через область уведомлений
· Флаги окна – настройка внешнего вида
· Таймер обновления – QTimer
· Обработка сигналов – корректное завершение

7. Процесс разработки.

· Сначала был создан базовый QWidget с кастомной отрисовкой аналоговых часов через QPainter.
· Затем добавлен QTimer для автоматического обновления времени с частотой 60 кадров в секунду.
· Реализована обработка событий мыши для перетаскивания окна часов.
· Добавлены флаги окна для создания прозрачного окна без рамки (FramelessWindowHint, WA_TranslucentBackground).
· Создан класс MainWindow и добавлена интеграция с QSystemTrayIcon.
· Разработано контекстное меню трея с действиями для управления приложением.
· Реализована функция переключения режима "поверх всех окон".
· Добавлена обработка сигналов SIGINT и SIGTERM для корректного завершения.
· Проведено тестирование под различными оконными менеджерами и операционными системами.

8. Основные сложности и способы их решения.

· (1) Сложность: при перетаскивании окно не перемещалось плавно или "дергалось".
  Решение: использованы глобальные координаты мыши вместо локальных. В mousePressEvent сохраняется globalPosition().toPoint(), а в mouseMoveEvent вычисляется смещение от этой позиции. Перемещение осуществляется через self.move() с новыми координатами.
· (2) Сложность: секундная стрелка двигалась дискретно (раз в секунду) из-за использования datetime.now().second.
  Решение: добавлены микросекунды в расчет: sec = now.second + now.microsecond / 1000000. Это позволяет стрелке двигаться плавно, пересчитывая позицию 60 раз в секунду.
· (3) Сложность: после переключения режима "поверх всех окон" окно становилось невидимым.
  Решение: при смене флагов окна необходимо вызвать self.clock_widget.show() после установки новых флагов, чтобы перерисовать окно с новыми параметрами.

9. Текущие ограничения проекта.

· Нет возможности изменения размера часов (размер фиксирован - 700×700).
· Отсутствует настройка цветовой схемы.
· Нет отображения даты.
· Приложение не сохраняет настройки (позиция окна, режим поверх всех окон) между сеансами.
· Отсутствует поддержка будильника или таймера.
· Нет горячих клавиш для быстрого управления.
· Иконка трея создается программно, без возможности загрузки пользовательской иконки.
· Приложение работает только под операционными системами с поддержкой Qt.
